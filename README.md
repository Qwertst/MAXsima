# MAXsima

Консольный peer-to-peer чат на Go с использованием gRPC двунаправленного стриминга.

## Требования

- Go 1.18+
- protoc (для регенерации proto)

## Установка

```bash
git clone https://github.com/aydreq/maxsima
cd maxsima
go mod download
```

## Сборка

```bash
make build
# или
go build -o maxsima ./cmd/chat
```

## Использование

### Режим сервера

Первый участник запускает сервер, указывая имя и порт:

```bash
./maxsima --username Alice --port 50051
```

### Режим клиента

Второй участник подключается к серверу:

```bash
./maxsima --username Bob --peer localhost:50051
```

После установки соединения оба участника могут отправлять сообщения в реальном времени. Каждое сообщение отображается в формате:

```
[2024-01-15 14:32:01] Alice: Hello!
```

Для выхода нажмите `Ctrl+C`.

## Флаги

| Флаг | Описание |
|------|----------|
| `--username` | Имя пользователя (обязательный) |
| `--port` | Порт для прослушивания (режим сервера) |
| `--peer` | Адрес peer-а для подключения (режим клиента) |

## Структура проекта

```
MAXsima/
├── cmd/chat/          # Точка входа
├── config/            # Конфигурация и парсинг флагов
├── internal/
│   ├── chat/          # Бизнес-логика: менеджер сессии
│   ├── client/        # gRPC-клиент
│   ├── model/         # Доменные модели
│   ├── server/        # gRPC-сервер
│   └── ui/            # Консольный интерфейс
├── proto/             # gRPC-контракт и сгенерированный код
└── Makefile
```

## Регенерация proto

```bash
make proto
```
